# Nyancat CLI

Nyancat rendered in telnet.

[![Nyancats](http://i.imgur.com/RMnUw.png)](http://i.imgur.com/RMnUw.png)

## Setup

Just run the npm install command:

    [sudo] npm install -g nyancat-telnet

To run the telnet server just type:

    nyancat-telnet
